# Support

OpenSyria Website is public and maintainer-led.

## Where to Ask

Use GitHub issues in this repository for:

- reproducible website bugs,
- accessibility issues,
- stale or unclear website content,
- broken links or incorrect page metadata,
- deployment, CI, tooling, or dependency maintenance,
- maintainer proposals for website behavior or structure.

Use `datasets-api` for:

- API endpoint behavior,
- OpenAPI or Scalar documentation issues,
- deployment, Docker, Redis, database, sync, or read-model import issues.

Use the relevant dataset repository for:

- incorrect records,
- missing records,
- source corrections,
- translation or alias corrections,
- dataset schema proposals.

For geography data, use the `data-geography` repository.

For university data, use the `data-universities` repository.

Public API documentation is available at <https://api.opensyria.org/docs>.
The production website is available at <https://opensyria.org>.

## Security

Do not report security vulnerabilities in public issues. Follow [SECURITY.md](SECURITY.md).

## Response Expectations

This project is maintained by a small maintainer team. Issues may be triaged by priority, project scope, reproducibility, accessibility impact, content safety, and release timing.

Maintainers may close issues that are out of scope, duplicates, missing reproduction details, unsafe to publish, or better handled in another repository.
